🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ):

1. Прочитайте:    QUICKSTART.md
2. Установите:    python setup.bat (или bash setup.sh)
3. Получите ключи:
   - Bot Token от @BotFather в Telegram
   - Google Spreadsheet ID
   - credentials.json от Google Cloud
4. Запустите:     python src/main.py
5. Выберите:      "1. ⚙️  Конфигурация"
6. Вводите ключи через интерактивное меню
7. Инициализируйте БД: python init_db.py
8. Запустите бота: python src/main.py → пункт 2
