🤖 Telegram Bot для Тату-Салона
═══════════════════════════════════════════════════════════════════════════════

Готовый к использованию Telegram бот для управления записями в тату-салон.

✨ ОСОБЕННОСТИ

✅ Интерактивная конфигурация через configure.py
✅ Работает с aiogram 2.15 (совместимость с Python 3.14)
✅ Интеграция с Google Sheets (база данных клиентов и бронирований)
✅ Интеграция с Google Calendar (расписание мастеров)
✅ Простой запуск: python run_bot.py
✅ Подробные инструкции и примеры

═══════════════════════════════════════════════════════════════════════════════

🚀 БЫСТРЫЙ СТАРТ (5 минут)

1. Конфигурация:
   python configure.py

2. Запуск бота:
   python run_bot.py

3. Откройте Telegram, найдите вашего бота, напишите /start

⚙️ КОНФИГУРАЦИЯ

Запустите для интерактивной конфигурации:
    python configure.py

Или отредактируйте .env файл вручную:
    TELEGRAM_BOT_TOKEN=<токен от @BotFather>
    ADMIN_IDS=<ваш ID из @userinfobot>
    GOOGLE_SPREADSHEET_ID=<ID Google Sheets>
    GOOGLE_CALENDAR_ID=<ID Google Calendar>
    SALON_NAME=<название салона>

🔐 ПОЛУЧЕНИЕ КЛЮЧЕЙ

1. TELEGRAM_BOT_TOKEN
   → Напиши @BotFather в Telegram
   → /newbot
   → Скопируй токен

2. ADMIN_ID
   → Напиши @userinfobot
   → Найди "Id: "
   → Скопируй номер

3. GOOGLE_SPREADSHEET_ID
   → https://sheets.google.com
   → Новая таблица
   → Из URL: docs.google.com/spreadsheets/d/{ID}/edit

4. GOOGLE_CALENDAR_ID
   → https://calendar.google.com
   → Параметры → ID календаря

❓ ПРОБЛЕМЫ И РЕШЕНИЯ

Проблема: ModuleNotFoundError: No module named 'aiogram'
────────────────────────────────────────────────────────
Решение: pip install -r requirements.txt

Проблема: Token is invalid!
─────────────────────────────
Решение: 
  - Запустите: python configure.py
  - Введите правильный токен от @BotFather

Проблема: Connection error
─────────────────────────────
Решение:
  - Проверьте интернет соединение
  - Если доступ заблокирован, используйте VPN
