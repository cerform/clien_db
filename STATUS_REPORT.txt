🚀 КАК НАЧАТЬ ИСПОЛЬЗОВАТЬ

ВАРИАНТ 1 (самый простой - меню):
  1. Откройте PowerShell в папке проекта
  2. .\start.ps1
  3. Выберите "1. Конфигурация"
  4. Введите токен от @BotFather
  5. Выберите "2. Запустить бота"
  6. Готово! Бот работает

ВАРИАНТ 2 (быстрый - командная строка):
  1. python configure.py       (ввод токена)
  2. python run_bot.py         (запуск бота)

ВАРИАНТ 3 (ручной - редактирование файла):
  1. Откройте .env в текстовом редакторе
  2. Найдите TELEGRAM_BOT_TOKEN=
  3. Вставьте ваш токен
  4. Сохраните файл
  5. python run_bot.py

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

Q: Бот не запускается - что делать?
A: 1. Проверьте токен (python configure.py)
   2. Проверьте интернет
   3. Посмотрите логи: type logs\bot.log

Q: Что такое токен?
A: Уникальный идентификатор вашего бота для Telegram API
   Формат: 123456789:ABCDEFGHIJKLMNOPQRSTUVWXYZabc

Q: Где .env файл?
A: В корне проекта (рядом с run_bot.py)
   Можно открыть любым текстовым редактором

Q: Нужно ли устанавливать Google API?
A: Нет, основной функционал работает без них
   Нужны только для сохранения данных в Google Sheets

Q: Могу ли я удалить .env?
A: Нет, он нужен для работы бота
   Содержит важные переменные конфигурации

📞 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

Проблема: ModuleNotFoundError: No module named 'aiogram'
Решение:  pip install aiogram

Проблема: Token is invalid!
Решение:  python configure.py (ввести правильный токен)

Проблема: Connection error
Решение:  Проверить интернет, может нужен VPN

Проблема: Port already in use
Решение:  Закрыть другой экземпляр бота

Проблема: Command not found: .\start.ps1
Решение:  powershell -ExecutionPolicy Bypass -File .\start.ps1
