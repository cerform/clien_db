2. ЕСЛИ ТОКЕНА НЕТ:
   └─ Откройте Telegram
   └─ Найдите @BotFather
   └─ Напишите: /newbot
   └─ Следуйте инструкциям
   └─ Скопируйте токен
   └─ Запустите configure.py
   └─ Вставьте токен

3. ПРОВЕРЬТЕ ИНТЕРНЕТ:
   └─ Убедитесь, что интернет работает
   └─ Если блокировка → используйте VPN

❌ ЕСЛИ ЕСТЬ ОШИБКИ

Error: Token is invalid!
└─ Проверьте токен в .env
└─ Убедитесь, что нет пробелов в начале/конце
└─ Возьмите новый токен у @BotFather

Error: This event loop is already running
└─ Используйте cmd.exe вместо PowerShell
└─ Или используйте .\start.ps1

Connection timeout
└─ Проверьте интернет
└─ Попробуйте VPN
└─ Проверьте брандмауэр

ModuleNotFoundError
└─ Установите: pip install -r requirements.txt
└─ Проверьте: pip list | grep aiogram
