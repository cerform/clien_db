❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

Q: Какие нужны токены?
A: На этом этапе нужен только TELEGRAM_BOT_TOKEN
   Остальные (Google API) опциональны

Q: Где открыть PowerShell?
A: Shift + Right Click в папке → "Open PowerShell window here"

Q: Токен потеря/забыл?
A: Напишите @BotFather снова и создайте новый токен
   Старый перестанет работать

Q: Я не знаю мой ID в Telegram?
A: Напишите @userinfobot любое сообщение, он пришлет ваш ID

Q: Что если бот не запускается?
A: Проверьте:
   1. Интернет подключение
   2. Токен в .env правильный (пробелы в начале/конце?)
   3. Python работает: python --version

Q: Можно ли запустить бота на сервере?
A: Да, но нужно использовать Webhook вместо polling
   Смотрите src/main.py для примера

📞 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК

Ошибка: ModuleNotFoundError: No module named 'aiogram'
  → pip install -r requirements.txt

Ошибка: Token is invalid!
  → Проверьте токен в .env (можно вводить через configure.py)

Ошибка: Connection error
  → Проверьте интернет, может нужен VPN

Ошибка: Port already in use
  → Закройте другие экземпляры бота или смените порт

Другие ошибки:
  → Смотрите полный лог: type logs\bot.log

💡 ПОЛЕЗНЫЕ СОВЕТЫ

1. Сохраняйте .env в безопасном месте (не в Git!)
2. Не делитесь токеном с никем
3. Регулярно обновляйте зависимости: pip install -r requirements.txt --upgrade
4. Используйте логи для отладки: смотрите папку logs/
5. Тестируйте команды в Telegram перед развертыванием на сервер
