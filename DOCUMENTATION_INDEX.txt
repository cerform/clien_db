🔧 ИСПОЛЬЗОВАНИЕ ОСНОВНЫХ СКРИПТОВ

run_bot.py
──────────
Запуск бота. Использование:
  python run_bot.py

Что происходит:
  1. Читает TELEGRAM_BOT_TOKEN из .env
  2. Инициализирует бота
  3. Устанавливает команды /start, /help, /about
  4. Запускает polling (ожидание сообщений)

Выход: Ctrl+C


configure.py
────────────
Интерактивная конфигурация. Использование:
  python configure.py

Что происходит:
  1. Спрашивает TELEGRAM_BOT_TOKEN
  2. Спрашивает ADMIN_IDS
  3. Спрашивает GOOGLE_SPREADSHEET_ID (опционально)
  4. Спрашивает GOOGLE_CALENDAR_ID (опционально)
  5. Спрашивает SALON_NAME
  6. Сохраняет в .env

Выход: Enter после завершения


start.ps1
─────────
Меню запуска для PowerShell. Использование:
  .\start.ps1

Меню:
  1 = Конфигурация (запусти configure.py)
  2 = Запуск бота (запусти run_bot.py)
  3 = Показать инструкцию
  4 = Выход

Выход: Нажми 4


start.bat
─────────
Меню запуска для CMD. Использование:
  start.bat

Меню: (аналогично start.ps1)
  1 = Конфигурация
  2 = Запуск бота
  3 = Инструкция
  4 = Выход
