🎯 СЛЕДУЮЩИЕ ШАГИ

Шаг 1: Получить TELEGRAM_BOT_TOKEN
  → Telegram → @BotFather → /newbot → скопировать токен

Шаг 2: Конфигурировать бота
  → python configure.py
  → Вставить токен
  → Вставить ваш ID (из @userinfobot)

Шаг 3: Запустить бота
  → python run_bot.py

Шаг 4: Проверить в Telegram
  → Найти бота
  → Написать /start
  → Если ответит - ВСЁ РАБОТАЕТ! 🎉
