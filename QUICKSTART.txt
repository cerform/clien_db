🔐 ПОЛУЧЕНИЕ НЕОБХОДИМЫХ КЛЮЧЕЙ И ID

1️⃣ TELEGRAM_BOT_TOKEN
──────────────────────
  1. Откройте чат с @BotFather в Telegram
  2. Напишите: /newbot
  3. Дайте название боту
  4. Скопируйте токен формата: 123456789:ABCDEFGHIJKLMNOPQRSTUVWXYZabcd
  5. Вставьте в configure.py

2️⃣ ADMIN_IDS
────────────
  1. Откройте чат с @userinfobot в Telegram
  2. Найдите строку "Id: "
  3. Скопируйте цифровой ID
  4. Вставьте в configure.py

3️⃣ GOOGLE_SPREADSHEET_ID
────────────────────────
  1. Откройте Google Sheets: https://sheets.google.com
  2. Создайте новую таблицу или откройте существующую
  3. Скопируйте ID из URL:
     https://docs.google.com/spreadsheets/d/{ID_ЭТО_ОН}/edit
  4. Вставьте в configure.py

4️⃣ GOOGLE_CALENDAR_ID
─────────────────────
  1. Откройте Google Calendar: https://calendar.google.com
  2. Нажмите на название календаря → Параметры
  3. Найдите "ID календаря"
  4. Скопируйте значение
  5. Вставьте в configure.py

🔧 ВОЗМОЖНЫЕ ПРОБЛЕМЫ

❓ "ModuleNotFoundError: No module named 'aiogram'"
──────────────────────────────────────────────────
  Решение: Установите зависимости еще раз
    pip install -r requirements.txt

❓ "ValidationError: Token is invalid!"
──────────────────────────────────────
  Решение: Проверьте, что токен в .env правильный
    - Запустите: python configure.py
    - Введите правильный токен из @BotFather

❓ "Connection error" или "Socket error"
────────────────────────────────────────
  Решение: Проверьте интернет соединение
    - Либо используйте VPN если доступ заблокирован
